# Consumer-Return Latency & the Cyclic Consumer Economy

An open-source viewpoint + working-paper + runnable model proposing that the **latency of
the consumer-return loop** — how fast created value cycles back into broad consumer
purchasing power — is the control parameter that decides whether an economy **diverges
(a "K-shape")** or settles into a **bounded rising oscillation (a "sine-on-a-trend")**.

> One-line thesis: *Hold money creation fixed (it enters at the top, per Cantillon). Then
> the speed of the return loop to consumers decides the regime. Slow loop → K. Fast loop → wave.*

## The claim in three pictures

1. **K-shape** — weak/slow consumer return → saddle → the top/bottom gap forks open.
2. **The funnel** — leverage outruns leakage → unstable spiral → cycle amplitude explodes (Minsky).
3. **The target** — fast consumer return → stable spiral on a trend → bounded, rising, shared cycle.

The central result: add a return **delay** $\tau$ and a stable economy crosses a **Hopf threshold**
$\tau^*$ into divergence. **Latency is the bifurcation parameter.**

## How this relates to prior work (read before citing)

This is a **synthesis-plus-lever**, not a new economic law. It extends: Cantillon (money-injection
asymmetry), Goodwin (predator-prey growth cycle), Minsky (leverage instability), Piketty ($r>g$),
and the consumer-stake/dividend family (Varoufakis UBD, Yang, Altman, Alaska Permanent Fund,
the web3 "ownership economy"). The new edge is **latency-as-control-parameter** and a
**velocity-first** instrument (shorten the loop) rather than a fairness-first, slow-payout one.
See §4 and §10 of the white paper.
